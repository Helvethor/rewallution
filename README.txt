Description du jeu :

Rewallution est un mini jeu multijoueur sur navigateur en 2D. Il utilise Air Console.
Chaque joueur utilise son smartphone comme manette de jeu.
Un des joueurs, déterminé au hasard, jouera contre tous les autres.
Le joueur choisi est Trump. Il se trouvera en haut de l'écran. Il aura la possibilité de tourner des blocs de briques de différentes formes, puis les lancera sur les autres joueurs situés en dessous.
Les autres joueurs seront les Mexicains. Ils devront éviter les blocs de briques, et grimper dessus pour atteindre le haut du mur, où se situe Trump. Ils peuvent sauter et changer de direction.
Tous les joueurs se déplacent automatiquement en aller-retour. C'est un jeu de coordination très rapide.
Différents bonus peuvent apparaître n'importe où sur le jeu pour ceux qui essaient de survivre. Si la partie dure trop longtemps, les Mexicains  qui vivent encore l'emportent.
Appréciez tout particulièrement la bande-son ainsi que le design graphique.

Amusez-vous bien !

Pour jouer :

Lancer votre navigateur, et allez sur le site https://airconsole.com/#http://shabang.ch/
Lancez l'application AirConsole sur votre smartphone, ou accédez au site airconsole.ch.
Liez votre smartphone au jeu en mettant le code donné su le navigateur.
Une fois que les participants sont prêts, appuyez sur le bouton sur votre smartphone pour lancer le jeu.

Commandes de jeu :

Trump : Bouton pour faire tourner le bloc de 90°, bouton pour faire tomber le bloc.
Mexicain : Bouton pour sauter, bouton pour changer de direction.

Matériel :
1 ordinateur avec navigateur, autant de smartphone que de joueurs.
